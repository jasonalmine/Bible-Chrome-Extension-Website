# Bible Verse New Tab - Landing Page

Marketing website for the Bible Verse New Tab Chrome extension.

## Live Site

[https://bible-verse-newtab.vercel.app](https://bible-verse-newtab.vercel.app)

## Tech Stack

- Static HTML/CSS/JS
- No build step required
- Deployed on Vercel

## Development

Open `index.html` in your browser, or use a local server:

```bash
# Using Python
python -m http.server 3000

# Using Node.js
npx serve
```

## Deployment

Push to GitHub and connect to Vercel, or deploy manually:

```bash
vercel
```

## Project Structure

```
bible-chrome-extension-website/
├── index.html          # Main landing page
├── privacy.html        # Privacy policy page
├── vercel.json         # Vercel configuration
├── css/
│   └── styles.css      # All styles
├── js/
│   └── main.js         # Animations & interactions
└── images/
    ├── favicon.svg     # Site favicon
    ├── hero-bg.jpg     # Hero background (add your own)
    ├── screenshot.png  # Extension screenshot (add your own)
    └── og-image.png    # Social sharing image (add your own)
```

## Required Images

You need to add these images:

1. **hero-bg.jpg** - A beautiful nature/galaxy background for the hero section
2. **screenshot.png** - Screenshot of the extension in action (1200x750 recommended)
3. **og-image.png** - Social sharing image (1200x630 recommended)

## Design

- **Colors**: Deep blue (#1e3a5f) + warm gold (#d4a574)
- **Fonts**: Playfair Display (headings) + Inter (body)
- **Style**: Spiritual, peaceful, modern
