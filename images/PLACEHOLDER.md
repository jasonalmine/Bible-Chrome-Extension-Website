# Required Images

Please add the following images to complete the website:

## 1. screenshot.png
- Screenshot of the extension in action
- Recommended size: 1200x750 pixels
- Shows a Bible verse on a beautiful background

## 2. og-image.png (optional)
- Social media sharing image
- Recommended size: 1200x630 pixels
- Shows extension name and a preview

## How to take a screenshot:
1. Load the extension in Chrome
2. Open a new tab
3. Use Chrome DevTools (Cmd+Shift+P > "Capture full size screenshot")
4. Save as screenshot.png in this folder
